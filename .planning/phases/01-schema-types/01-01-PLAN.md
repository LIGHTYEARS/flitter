## Message 判别联合 (role)

```typescript
type Message = UserMessage | AssistantMessage | InfoMessage;
```

## UserMessage
```typescript
interface UserMessage {
  role: "user";
  messageId: number;
  content: UserContentBlock[];
  dtwMessageID?: string;
  interrupted?: boolean;
  fileMentions?: FileMentions;
  userState?: UserState;
  discoveredGuidanceFiles?: string[];
  agentMode?: string;
  meta?: MessageMeta;
  readAt?: number;
  originalToolUseInput?: Record<string, unknown>;
  parentToolUseId?: string;
}
```

## AssistantMessage
```typescript
interface AssistantMessage {
  role: "assistant";
  messageId: number;
  content: AssistantContentBlock[];
  state: MessageState;
  usage?: Usage;
  dtwMessageID?: string;
  readAt?: number;
  nativeMessage?: NativeMessage;
  meta?: { openAIResponsePhase?: "commentary" | "final_answer" };
  parentToolUseId?: string;
}
```

## InfoMessage
```typescript
interface InfoMessage {
  role: "info";
  messageId: number;
  content: InfoContentBlock[];
  dtwMessageID?: string;
}
```

## User Content Blocks (type 判别)
```typescript
// TextBlock
{ type: "text"; text: string; cache_control?: { type: "ephemeral"; ttl: string } }

// ImageBlock
{ type: "image"; source: Base64Source | URLSource; sourcePath?: string; cache_control?: CacheControl }
// Base64Source: { type: "base64"; mediaType: string; data: string }
// URLSource: { type: "url"; url: string }

// ToolResultBlock
{ type: "tool_result"; toolUseID: string; run: ToolRun; userInput?: unknown; cache_control?: CacheControl }
```

## Assistant Content Blocks (type 判别)
```typescript
// TextContentBlock
{ type: "text"; text: string; startTime?: number; finalTime?: number }

// ToolUseBlock
{ type: "tool_use"; id: string; name: string; complete: boolean; input: Record<string, unknown>;
  inputPartialJSON?: { json: string }; inputIncomplete?: Record<string, unknown>;
  startTime?: number; finalTime?: number; metadata?: { thoughtSignature?: string } }

// ThinkingBlock
{ type: "thinking"; thinking: string; signature: string;
  provider?: "anthropic" | "vertexai" | "openai";
  openAIReasoning?: { id: string; encryptedContent: string };
  startTime?: number; finalTime?: number }

// RedactedThinkingBlock
{ type: "redacted_thinking"; data: string; provider?: "anthropic" | "vertexai" | "openai";
  startTime?: number; finalTime?: number }

// ServerToolUseBlock
{ type: "server_tool_use"; id: string; name: string; input: unknown;
  startTime?: number; finalTime?: number }
```

## Info Content Blocks (type 判别)
```typescript
// ManualBashInvocationBlock
{ type: "manual_bash_invocation"; args: { cmd: string; cwd?: string }; toolRun: ToolRun; hidden?: boolean }

// SummaryBlock
{ type: "summary"; summary: { type: "message"; summary: string } }
```

## MessageState (type 判别)
```typescript
type MessageState =
  | { type: "streaming" }
  | { type: "complete"; stopReason: "end_turn" | "tool_use" | "max_tokens" }
  | { type: "cancelled" }
  | { type: "error"; error: { message: string } };
```

## ToolRun (status 判别)
```typescript
type ToolRun =
  | { status: "done"; result?: unknown; progress?: unknown; trackFiles?: string[] }
  | { status: "error"; error: { message: string; errorCode?: string } }
  | { status: "cancelled"; reason?: string }
  | { status: "rejected-by-user"; reason?: string }
  | { status: "in-progress"; progress?: unknown };
```

## Usage
```typescript
interface Usage {
  model: string;
  maxInputTokens: number;
  inputTokens: number;
  outputTokens: number;
  cacheCreationInputTokens: number | null;
  cacheReadInputTokens: number | null;
  totalInputTokens: number;
  thinkingBudget?: number;
  timestamp: string; // ISO 8601
}
```

## MessageMeta
```typescript
interface MessageMeta {
  sentAt?: number;
  fromAggman?: true;
  fromExecutorThreadID?: string;
}
```

## FileMentions
```typescript
interface FileMentions {
  files: Array<{ uri: string; content: string; isImage: boolean;
    imageInfo?: { mimeType: string; size: number } }>;
  mentions: Array<{ uri: string }>;
  imageBlocks?: ImageBlock[];
}
```

  <name>Task 2: 定义所有 LLM 消息类型 Zod Schema</name>
  <files>packages/schemas/src/messages.ts</files>
  <read_first>
    - packages/schemas/src/index.ts
  </read_first>
  <action>
创建 `packages/schemas/src/messages.ts`，按以下顺序定义 Zod schema:

**基础类型:**
- `CacheControlSchema` = z.object({ type: z.literal("ephemeral"), ttl: z.string() })
- `Base64SourceSchema` = z.object({ type: z.literal("base64"), mediaType: z.string(), data: z.string() })
- `URLSourceSchema` = z.object({ type: z.literal("url"), url: z.string() })
- `ImageSourceSchema` = z.discriminatedUnion("type", [Base64SourceSchema, URLSourceSchema])

**Usage:**
- `UsageSchema` 包含 model, maxInputTokens, inputTokens, outputTokens, cacheCreationInputTokens (nullable), cacheReadInputTokens (nullable), totalInputTokens, thinkingBudget (optional), timestamp

**MessageMeta:**
- `MessageMetaSchema` 包含 sentAt (optional number), fromAggman (optional literal true), fromExecutorThreadID (optional string)

**ToolRun (status 判别联合):**
- 5 种 status 变体: done, error, cancelled, rejected-by-user, in-progress

**MessageState (type 判别联合):**
- 4 种 type 变体: streaming, complete, cancelled, error

**User Content Blocks (type 判别联合):**
- TextBlock, ImageBlock, ToolResultBlock

**Assistant Content Blocks (type 判别联合):**
- TextContentBlock, ToolUseBlock, ThinkingBlock, RedactedThinkingBlock, ServerToolUseBlock

**Info Content Blocks (type 判别联合):**
- ManualBashInvocationBlock (含嵌套 ToolRun), SummaryBlock

**FileMentions:**
- files 数组, mentions 数组, imageBlocks 可选

**Messages (role 判别联合):**
- UserMessageSchema, AssistantMessageSchema, InfoMessageSchema
- MessageSchema = z.discriminatedUnion("role", [...])

**导出:**
- 所有 Schema + 所有 inferred 类型 (z.infer)
- 更新 `src/index.ts` 重新导出 `./messages`
