## PermissionMatcher (递归联合)
```typescript
// 从 Zod z.lazy() 实现
type PermissionMatcher =
  | string                               // Glob 模式: "*" 匹配任意，"/regex/" 正则，精确字符串
  | PermissionMatcher[]                   // OR 匹配 (min 1 element)
  | boolean                              // 字面量布尔匹配
  | number                               // 字面量数字匹配
  | null                                 // 字面量 null 匹配
  | undefined                            // 字面量 undefined 匹配
  | Record<string, PermissionMatcher>;    // 对象键深度匹配
```

## PermissionEntry (v2 Zod schema)
```typescript
interface PermissionEntry {
  tool: string;                                            // 必填，min length 1
  matches?: Record<string, PermissionMatcher>;             // 可选，工具输入条件匹配
  action: "allow" | "reject" | "ask" | "delegate";        // 必填
  context?: "thread" | "subagent";                         // 可选
  to?: string;                                             // action="delegate" 时必填，min length 1
  message?: string;                                        // action="reject" 时可用
}

// 约束:
// - action === "delegate" → "to" 必填
// - action !== "delegate" → "to" 禁止
// - "message" 仅在 action === "reject" 时有效
```
