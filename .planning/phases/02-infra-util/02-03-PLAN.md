## GitStatusSnapshot

```typescript
interface GitStatusSnapshot {
  provider: "git";
  capturedAt: number;          // Date.now()
  available: boolean;
  repositoryRoot: string;
  repositoryName: string;
  branch: string | null;
  head: string | null;         // HEAD SHA
  baseRef: string | null;
  baseRefHead: string | null;
  aheadCount: number;
  behindCount: number | undefined;
  aheadCommits: AheadCommit[];
  files: GitFileChange[];
}
```

## 内部解析

```typescript
// git status --porcelain=v1 -z 输出解析
// 格式: XY PATH\0 (renamed: XY OLD_PATH\0NEW_PATH\0)
// X = index status, Y = work-tree status
// 状态码映射:
//   M = modified, A = added, D = deleted, R = renamed, ?? = untracked
//   ' ' (空格) = 未变更
function parsePortalainStatus(output: string): StatusEntry[];

interface StatusEntry {
  x: string;       // index status
  y: string;       // work-tree status
  path: string;
  origPath?: string; // for renames
}
```

创建 `packages/util/src/git/git.ts`，实现:

**内部工具函数:**
- `parsePortalainStatus(output: string): StatusEntry[]`
  - 解析 `git status --porcelain=v1 -z` 输出
  - NUL (\0) 分隔的条目
  - 前两字符为 XY 状态码，第三字符为空格，之后为路径
  - Renamed 条目有两个路径 (旧路径\0新路径)
  - 状态码映射: M→modified, A→added, D→deleted, R→renamed, ??→untracked

- `parseDiffStat(diffOutput: string): DiffStat`
  - 解析 `git diff --numstat` 输出
  - 格式: `additions\tdeletions\tpath`

**公共 API:**

1. `isGitRepository(path: string): Promise<boolean>`
   - 执行 `git rev-parse --is-inside-work-tree`
   - exitCode 0 → true, 否则 → false

2. `getCurrentBranch(repoRoot: string): Promise<string | null>`
   - 执行 `git symbolic-ref --short HEAD`
   - 成功 → 分支名, 失败 (detached HEAD) → null

3. `getGitDiff(repoRoot: string, ref?: string): Promise<string>`
   - 无 ref: `git diff HEAD`
   - 有 ref: `git diff ref`
   - 返回 diff 字符串

4. `captureGitStatus(workspaceRoot, options?): Promise<GitStatusSnapshot>`
   聚合以下 git 命令结果:
   - `git rev-parse --show-toplevel` → repositoryRoot
   - `git rev-parse --short HEAD` → head
   - `git symbolic-ref --short HEAD` → branch
   - `git status --porcelain=v1 -z` → files (通过 parsePortalainStatus)
   - `git log --oneline -n 20 origin/{branch}..HEAD` → aheadCommits (如有 remote)
   - `git rev-list --count HEAD...origin/{branch}` → aheadCount/behindCount (如有 remote)
   - `git diff HEAD -- {path}` → 每个变更文件的 diff
   - `git diff --numstat HEAD` → diffStat
   - repositoryName 从 repositoryRoot 最后一段路径提取
   - maxDiffBufferBytes 限制单文件 diff 大小 (默认 50KB)
   - 任何 git 命令失败时设 available=false 并返回降级快照
