## VtParser

```typescript
class VtParser {
  /** 解析字节流，通过回调发射 VtEvent */
  parse(data: Buffer | Uint8Array): void;

  /** 注册事件回调 */
  onEvent(handler: (event: VtEvent) => void): void;

  /** 重置到初始状态 */
  reset(): void;

  /** 获取当前状态 (测试用) */
  getState(): VtParserState;
}

type VtParserState =
  | "ground"
  | "escape"
  | "escape_intermediate"
  | "csi_entry"
  | "csi_param"
  | "csi_intermediate"
  | "csi_ignore"
  | "osc_string"
  | "dcs_entry"
  | "dcs_param"
  | "dcs_intermediate"
  | "dcs_passthrough"
  | "dcs_ignore"
  | "apc_string"
  | "sos_string"
  | "pm_string";
```

## 状态机转换规则

```
Ground:
  0x00-0x17, 0x19, 0x1C-0x1F → execute (C0 control)
  0x20-0x7E → print (可打印 ASCII)
  0x80-0xFF → UTF-8 accumulate (多字节序列)
  0x1B → Escape state
  0x9B → CSI_Entry (8-bit CSI)
  0x9D → OSC_String (8-bit OSC)
  0x90 → DCS_Entry (8-bit DCS)
  0x9E → PM_String
  0x9F → APC_String

Escape:
  0x20-0x2F → Escape_Intermediate (collect intermediates)
  0x30-0x4F, 0x51-0x57, 0x59, 0x5A, 0x5C, 0x60-0x7E → esc_dispatch → Ground
  0x5B ('[') → CSI_Entry
  0x5D (']') → OSC_String
  0x50 ('P') → DCS_Entry
  0x58 ('X') → SOS_String
  0x5E ('^') → PM_String
  0x5F ('_') → APC_String

CSI_Entry:
  0x30-0x39, 0x3B → CSI_Param (param bytes)
  0x3C-0x3F → collect private marker, CSI_Param
  0x20-0x2F → CSI_Intermediate
  0x40-0x7E → csi_dispatch → Ground

CSI_Param:
  0x30-0x39 → accumulate digit
  0x3A → subparam separator (colon)
  0x3B → param separator (semicolon)
  0x20-0x2F → CSI_Intermediate
  0x40-0x7E → csi_dispatch → Ground
  0x3C-0x3F → CSI_Ignore (invalid)

OSC_String:
  0x20-0x7E, 0x80-0xFF → accumulate OSC data
  0x07 (BEL) → osc_dispatch → Ground
  0x1B → check for ST (ESC \)
  0x9C (ST) → osc_dispatch → Ground

DCS (similar structure to CSI + passthrough data)
APC/SOS/PM (accumulate until ST)
```

   ```
   state, params[], paramBuffer, subparamBuffer, currentSubparams[],
   intermediates[], private_marker[], oscData[], dcsData[], apcData[],
   printBuffer (Uint8Array), textBuffer (string), escSeen flags
   ```

2. 状态机实现:
   - Ground 状态: ASCII 可打印 → print, 0x1B → Escape, C0 控制码处理
   - Escape 状态: `[` → CSI, `]` → OSC, `P` → DCS, `_` → APC, 其他 → esc_dispatch
   - CSI 状态链: Entry → Param (数字/分号/冒号) → Intermediate → dispatch
   - OSC 状态: 累积数据直到 BEL (0x07) 或 ST (ESC \)
   - DCS 状态链: Entry → Param → Intermediate → Passthrough → unhook
   - APC/SOS/PM 状态: 累积数据直到 ST

3. UTF-8 解码:
   - printBuffer 累积字节
   - 使用 TextDecoder("utf-8", { fatal: false }) 解码
   - 用 Intl.Segmenter("en", { granularity: "grapheme" }) 分割 grapheme cluster
   - 每个 grapheme 作为单独 VtPrintEvent 发射

4. CSI 参数解析:
   - 分号 (;) 分隔主参数
   - 冒号 (:) 分隔子参数
   - 缺省值为 -1
   - 溢出参数忽略

5. `parse(data)` — 逐字节驱动状态机
6. `onEvent(handler)` — 注册回调
7. `reset()` — 重置所有状态到 ground
8. `getState()` — 返回当前状态 (测试用)
