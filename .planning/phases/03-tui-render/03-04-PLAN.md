## InputParser

```typescript
class InputParser {
  /** 处理 VtEvent，通过回调发射 InputEvent */
  handleVtEvent(event: VtEvent): void;

  /** 注册输入事件回调 */
  onInput(handler: (event: InputEvent) => void): void;
}
```

## CSI Final Byte → Key Mapping

```
A → ArrowUp       B → ArrowDown
C → ArrowRight     D → ArrowLeft
H → Home           F → End
P → F1 (SS3)       Q → F2 (SS3)
R → F3 (SS3)       S → F4 (SS3)
Z → Shift+Tab (backtab)
```

## CSI Tilde (~) Key Mapping

```
CSI param ~ mapping:
1~  → Home         2~  → Insert
3~  → Delete       4~  → End
5~  → PageUp       6~  → PageDown
7~  → Home (alt)   8~  → End (alt)
11~ → F1           12~ → F2
13~ → F3           14~ → F4
15~ → F5           17~ → F6
18~ → F7           19~ → F8
20~ → F9           21~ → F10
23~ → F11          24~ → F12
```

## Modifier Bitmask (CSI param 2)

```
CSI 1;{mod}A format:
mod = 1 + bitmask where:
  bit 0 (1) = Shift
  bit 1 (2) = Alt
  bit 2 (4) = Ctrl
  bit 3 (8) = Meta

So: 2=Shift, 3=Alt, 4=Shift+Alt, 5=Ctrl, 6=Ctrl+Shift, 7=Ctrl+Alt, 8=Ctrl+Shift+Alt, 9=Meta
```

## Ctrl+Letter Mapping

```
0x01 → Ctrl+a, 0x02 → Ctrl+b, ..., 0x1A → Ctrl+z
0x00 → Ctrl+Space (or Ctrl+@)
0x1B → Escape
0x7F → Backspace
0x08 → Backspace (alt)
0x09 → Tab
0x0A/0x0D → Enter
```

## SGR Mouse Format

```
CSI < button ; x ; y M  → mouse press
CSI < button ; x ; y m  → mouse release

button encoding:
0 = left press
1 = middle press
2 = right press
32+ = motion flag (add 32 for move events)
64 = wheel up
65 = wheel down
```

   - SGR 鼠标 (private="<"): 解析 button/x/y, final "M"=press "m"=release
   - Tilde 映射 (final="~"): param[0] 查表得到 key, param[1] 解码 modifier
   - 方向键/Home/End/功能键: final 查表, param[1] 解码 modifier
   - CSI Z → Shift+Tab
   - CSI I / CSI O → FocusEvent
   - CSI 200~ → 开始 paste, CSI 201~ → 结束 paste

4. VtEscapeEvent 处理:
   - SS3 (intermediates="O"): P=F1, Q=F2, R=F3, S=F4, A-D=方向键
   - 其他 escape → KeyEvent with alt modifier

5. Ctrl+字母处理 (通过 print event char code < 0x20):
   - 0x01-0x1A → key="a"-"z" + ctrl
   - 0x00 → key="Space" + ctrl (or key="@" + ctrl)
   - 0x1B → key="Escape"
   - 0x7F → key="Backspace"
   - 0x09 → key="Tab"
   - 0x0A/0x0D → key="Enter"
