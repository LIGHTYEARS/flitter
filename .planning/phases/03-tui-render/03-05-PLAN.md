## ScreenBuffer (from pY 类)

```typescript
class ScreenBuffer {
  readonly width: number;
  readonly height: number;

  constructor(width: number, height: number);

  /** 获取 Cell (越界返回 Cell.EMPTY) */
  getCell(x: number, y: number): Cell;

  /** 设置 Cell (越界忽略) */
  setCell(x: number, y: number, cell: Cell): void;

  /** 写入字符到位置 (自动创建 Cell) */
  writeChar(x: number, y: number, char: string, style: TextStyle, width?: number): void;

  /** 清空所有 Cell 为 EMPTY */
  clear(): void;

  /** 填充区域 */
  fill(x: number, y: number, w: number, h: number, cell: Cell): void;

  /** 调整大小 (保留现有内容) */
  resize(newWidth: number, newHeight: number): void;

  /** 复制内容到另一个 buffer (用于 present) */
  copyTo(target: ScreenBuffer): void;
}
```

## Screen (from Zx 类)

```typescript
class Screen {
  readonly width: number;
  readonly height: number;

  /** 前缓冲 (当前显示) */
  readonly front: ScreenBuffer;
  /** 后缓冲 (绘制目标) */
  readonly back: ScreenBuffer;

  /** 是否需要全屏刷新 */
  needsFullRefresh: boolean;

  /** 光标位置 (null=隐藏) */
  cursorPosition: { x: number; y: number } | null;
  cursorVisible: boolean;

  constructor(width: number, height: number);

  /** 获取后缓冲 Cell */
  getCell(x: number, y: number): Cell;

  /** 设置后缓冲 Cell + 标记 dirty */
  setCell(x: number, y: number, cell: Cell): void;

  /** 写字符到后缓冲 */
  writeChar(x: number, y: number, char: string, style: TextStyle, width?: number): void;

  /** 清空后缓冲 */
  clear(): void;

  /** 调整大小 */
  resize(newWidth: number, newHeight: number): void;

  /** 获取 dirty Cell 列表 [{x, y, cell}] + dirty 行标记 */
  getDirtyRegions(): DirtyRegion[];

  /** 交换前后缓冲，重置 dirty 标记 */
  present(): void;
}

interface DirtyRegion {
  y: number;
  cells: Array<{ x: number; cell: Cell }>;
}
```

创建 `packages/tui/src/screen/screen.ts`:

1. `Screen` 类:
   - `front` / `back`: 两个 ScreenBuffer
   - `dirtyRows: Set<number>` — 行级 dirty 标记
   - `dirtyCells: Map<number, Set<number>>` — Cell 级 dirty 标记 (row → set of cols)
   - `needsFullRefresh` 标志

2. `setCell(x, y, cell)`:
   - 如果 back.getCell(x,y).equals(cell) → 跳过 (不标记 dirty)
   - 否则: back.setCell + 标记 dirty (行+Cell)

3. `writeChar(x, y, char, style, width)`:
   - 调用 back.writeChar
   - 标记 dirty

4. `getDirtyRegions()`:
   - 如果 needsFullRefresh → 返回所有行所有列
   - 否则: 遍历 dirtyRows，每行收集 dirtyCells

5. `present()`:
   - back.copyTo(front) — 同步前后缓冲
   - 清空 dirtyRows + dirtyCells
   - needsFullRefresh = false

6. `resize(w, h)`:
   - front.resize + back.resize
   - needsFullRefresh = true (尺寸变化需全屏重绘)
   - 更新 width/height

7. `clear()`:
   - back.clear()
   - needsFullRefresh = true (或标记所有行 dirty)
  </action>
