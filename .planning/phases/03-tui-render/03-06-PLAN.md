## AnsiRenderer

```typescript
class AnsiRenderer {
  /** 从 Screen 生成差分 ANSI 输出 */
  render(screen: Screen): string;

  /** 生成全屏刷新 ANSI 输出 */
  renderFull(screen: Screen): string;

  /** 生成光标位置/可见性控制序列 */
  renderCursor(screen: Screen): string;
}
```

## ANSI 转义序列常量

```typescript
const ESC = "\x1b";
const CSI = ESC + "[";

// 光标控制
const CUP = (row: number, col: number) => `${CSI}${row + 1};${col + 1}H`;  // 绝对定位 (1-based)
const CUF = (n: number) => `${CSI}${n}C`;     // 右移
const CUB = (n: number) => `${CSI}${n}D`;     // 左移
const CUU = (n: number) => `${CSI}${n}A`;     // 上移
const CUD = (n: number) => `${CSI}${n}B`;     // 下移
const HIDE_CURSOR = `${CSI}?25l`;
const SHOW_CURSOR = `${CSI}?25h`;

// 屏幕控制
const CLEAR_SCREEN = `${CSI}2J`;
const CLEAR_LINE = `${CSI}2K`;

// SGR (Select Graphic Rendition)
const SGR = (...params: (string | number)[]) => `${CSI}${params.join(";")}m`;
const SGR_RESET = `${CSI}0m`;

// 备用屏幕
const ALT_SCREEN_ON = `${CSI}?1049h`;
const ALT_SCREEN_OFF = `${CSI}?1049l`;

// 鼠标模式
const MOUSE_ON = `${CSI}?1000h${CSI}?1003h${CSI}?1006h`;   // Button+Any+SGR
const MOUSE_OFF = `${CSI}?1000l${CSI}?1003l${CSI}?1006l`;

// Bracketed paste
const PASTE_ON = `${CSI}?2004h`;
const PASTE_OFF = `${CSI}?2004l`;
```

   - CLEAR_SCREEN, CLEAR_LINE
   - SGR, SGR_RESET
   - ALT_SCREEN_ON/OFF, MOUSE_ON/OFF, PASTE_ON/OFF

2. `AnsiRenderer` 类:

   `render(screen)`:
   - 如果 screen.needsFullRefresh → 调用 renderFull
   - 否则:
     a. 获取 dirtyRegions = screen.getDirtyRegions()
     b. 遍历每个 dirty 行:
        - 按 x 排序脏 Cell
        - 相邻 Cell 合并输出:
          - 移动光标到第一个脏 Cell (CUP)
          - 设置 SGR (使用 diffSgr 与上一个 Cell 的样式比较)
          - 输出字符
          - 如果下一个脏 Cell 不相邻 → CUP 跳转
        - 跳过 width=0 续位 Cell (不输出)
     c. 渲染光标
   - 返回完整 ANSI 字符串

   `renderFull(screen)`:
   - HIDE_CURSOR + CUP(0,0) + CLEAR_SCREEN
   - 逐行逐列输出所有非 EMPTY Cell
   - 合并相邻同样式 Cell
   - 恢复光标

   `renderCursor(screen)`:
   - 如果 cursorVisible + cursorPosition: SHOW_CURSOR + CUP
   - 否则: HIDE_CURSOR

3. 内部优化:
   - 跟踪 "当前光标位置" 避免不必要的 CUP
   - 跟踪 "当前样式" 使用 diffSgr 避免冗余 SGR
   - 跳过空行 (无脏 Cell)
