```typescript
/** 主轴对齐方式 */
type MainAxisAlignment = "start" | "end" | "center" | "spaceBetween" | "spaceAround" | "spaceEvenly";

/** 交叉轴对齐方式 */
type CrossAxisAlignment = "start" | "end" | "center" | "stretch";

/** 主轴尺寸策略 */
type MainAxisSize = "min" | "max";

/** 布局方向 */
type Axis = "horizontal" | "vertical";

/** 弹性适配模式 */
type FlexFit = "tight" | "loose";

/**
 * 弹性布局父数据，附加到 RenderFlex 的每个子节点上。
 *
 * @example
 * ```ts
 * const pd = new FlexParentData();
 * pd.flex = 2;
 * pd.fit = "tight";
 * ```
 */
class FlexParentData extends ParentData {
  /** 弹性因子，0 表示非弹性子节点 */
  flex: number;
  /** 弹性适配模式，tight 强制填满分配空间，loose 允许更小 */
  fit: FlexFit;
}

/**
 * 弹性布局渲染对象，实现 Flutter Flex 布局算法。
 *
 * 两遍布局流程:
 * 1. 第一遍: 布局非弹性子节点 (flex === 0)，累加主轴占用
 * 2. 计算剩余空间，按 flex 因子分配给弹性子节点
 * 3. 第二遍: 布局弹性子节点
 * 4. 计算交叉轴尺寸 (子节点最大值)
 * 5. 根据 mainAxisAlignment 定位所有子节点
 *
 * @example
 * ```ts
 * const flex = new RenderFlex({
 *   direction: "horizontal",
 *   mainAxisAlignment: "spaceBetween",
 *   crossAxisAlignment: "center",
 *   mainAxisSize: "max",
 * });
 * ```
 */
class RenderFlex extends RenderBox {
  /** 布局方向 */
  direction: Axis;
  /** 主轴对齐方式 */
  mainAxisAlignment: MainAxisAlignment;
  /** 交叉轴对齐方式 */
  crossAxisAlignment: CrossAxisAlignment;
  /** 主轴尺寸策略 */
  mainAxisSize: MainAxisSize;

  constructor(options: {
    direction: Axis;
    mainAxisAlignment?: MainAxisAlignment;   // 默认 "start"
    crossAxisAlignment?: CrossAxisAlignment; // 默认 "start"
    mainAxisSize?: MainAxisSize;             // 默认 "max"
  });

  /** 确保子节点拥有 FlexParentData */
  setupParentData(child: RenderObject): void;

  /** Flex 两遍布局算法 */
  performLayout(): void;
}
```

1. 导出类型别名:
   - `MainAxisAlignment` — `"start" | "end" | "center" | "spaceBetween" | "spaceAround" | "spaceEvenly"`
   - `CrossAxisAlignment` — `"start" | "end" | "center" | "stretch"`
   - `MainAxisSize` — `"min" | "max"`
   - `Axis` — `"horizontal" | "vertical"`
   - `FlexFit` — `"tight" | "loose"`

2. 导出 `FlexParentData extends ParentData`:
   - `flex: number = 0` — 弹性因子
   - `fit: FlexFit = "loose"` — 默认松适配

3. 导出 `RenderFlex extends RenderBox`:
   - **构造函数属性:**
     - `direction: Axis`
     - `mainAxisAlignment: MainAxisAlignment` (默认 `"start"`)
     - `crossAxisAlignment: CrossAxisAlignment` (默认 `"start"`)
     - `mainAxisSize: MainAxisSize` (默认 `"max"`)

   - **setupParentData(child)**:
     如果 `child.parentData` 不是 `FlexParentData` 实例，则设置 `child.parentData = new FlexParentData()`

   - **辅助方法** (private):
     - `_getMainAxisSize(size: Size): number` — 根据 direction 返回 width 或 height
     - `_getCrossAxisSize(size: Size): number` — 根据 direction 返回另一维度
     - `_getMainAxisConstraint(constraints: BoxConstraints): number` — 返回主轴方向的 maxWidth 或 maxHeight
     - `_getCrossAxisConstraint(constraints: BoxConstraints): number` — 返回交叉轴方向的 maxWidth 或 maxHeight

   - **performLayout()** — Flex 两遍布局算法:

     **第一遍: 布局非弹性子节点**
     1. 遍历 children，累加 `totalFlex` (来自 FlexParentData.flex)
     2. 对 `flex === 0` 的子节点:
        - 构造子约束: 主轴方向 loosen (0..maxMain)，交叉轴方向根据 crossAxisAlignment:
          - `"stretch"` → tight 约束 (min=max=crossAxis 可用空间)
          - 其他 → loose 约束 (0..crossMax)
        - 调用 `child.layout(childConstraints)`
        - 累加主轴已占用空间 `allocatedMainAxis`
        - 更新最大交叉轴尺寸 `crossAxisExtent`

     **计算剩余空间**
     3. `freeMainAxis = mainAxisConstraint - allocatedMainAxis` (clamp 到 0)
     4. `spacePerFlex = totalFlex > 0 ? freeMainAxis / totalFlex : 0`

     **第二遍: 布局弹性子节点**
     5. 对 `flex > 0` 的子节点:
        - `childMainAxis = spacePerFlex * flex`
        - 根据 fit:
          - `"tight"` → 主轴紧约束 `BoxConstraints.tight(childMainAxis, ...)`
          - `"loose"` → 主轴松约束 `0..childMainAxis`
        - 交叉轴同第一遍规则
        - 调用 `child.layout(childConstraints)`
        - 累加 `allocatedMainAxis`
        - 更新 `crossAxisExtent`

     **确定自身尺寸**
     6. 主轴尺寸:
        - `mainAxisSize === "max"` → 使用约束的 mainAxisConstraint
        - `mainAxisSize === "min"` → 使用 allocatedMainAxis
     7. 交叉轴尺寸: crossAxisExtent (clamp 到约束范围)
     8. 设置 `this.size`

     **根据 mainAxisAlignment 定位子节点**
     9. 计算起始偏移和间距:
        - `"start"` → offset=0, gap=0
        - `"end"` → offset=remainingSpace, gap=0
        - `"center"` → offset=remainingSpace/2, gap=0
        - `"spaceBetween"` → offset=0, gap=remainingSpace/(childCount-1)
        - `"spaceAround"` → gap=remainingSpace/childCount, offset=gap/2
        - `"spaceEvenly"` → gap=remainingSpace/(childCount+1), offset=gap
     10. 遍历 children，设置每个 child 的 offset:
        - direction=horizontal → offset = { x: mainOffset, y: crossOffset }
        - direction=vertical → offset = { x: crossOffset, y: mainOffset }
        - crossOffset 根据 crossAxisAlignment:
          - `"start"` → 0
          - `"end"` → crossAxisExtent - childCrossSize
          - `"center"` → (crossAxisExtent - childCrossSize) / 2
          - `"stretch"` → 0

4. JSDoc 中文注释: 每个类/方法/属性含功能说明
