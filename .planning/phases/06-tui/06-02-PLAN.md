创建 `packages/tui/src/scroll/list-view.ts`:

导出 `ListViewProps` 接口:
```typescript
interface ListViewProps {
  itemCount: number;
  itemBuilder: (index: number) => Widget;
  itemExtent?: number;         // 固定行高 (可选, 不提供则按需 layout)
  controller?: ScrollController;
  cacheExtent?: number;        // 视口外缓冲行数, 默认 5
}
```

导出 `ListView` (StatefulWidget 子类):
- State 字段:
  - `_controller: ScrollController` — 外部提供或内部创建
  - `_ownsController: boolean` — 是否自管理 controller
  - `_keyHandler: ScrollKeyHandler` — 键盘/鼠标处理
- `initState()`:
  - 如果外部未提供 controller, 创建新的 ScrollController, _ownsController=true
  - controller.addListener(() => setState())
- `dispose()`:
  - controller.removeListener
  - 如果 _ownsController, controller.dispose()

- `build()` 返回自定义 RenderObject Widget:
  - `performLayout(constraints)`:
    - viewportHeight = constraints.maxHeight
    - 如果 itemExtent 已知:
      - totalHeight = itemCount * itemExtent
      - firstVisibleIndex = Math.floor(controller.offset / itemExtent)
      - lastVisibleIndex = Math.min(itemCount - 1, Math.ceil((controller.offset + viewportHeight) / itemExtent))
      - 扩展缓冲: firstVisible - cacheExtent .. lastVisible + cacheExtent (clamp [0, itemCount-1])
    - 如果 itemExtent 未知:
      - 遍历 items, layout 每项, 累加高度, 记录每项偏移量
      - 通过二分搜索确定可见范围
    - 只调用 itemBuilder(i) 创建可见+缓冲范围内的 Widget
    - controller.updateMaxScrollExtent(max(0, totalHeight - viewportHeight))
    - 自身 size = constraints.constrain({ width: constraints.maxWidth, height: viewportHeight })
  - `paint(screen, offset)`:
    - 按 offset 偏移渲染可见子项
