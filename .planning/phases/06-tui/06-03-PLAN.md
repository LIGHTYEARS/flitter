导出 `TextLayoutEngine` 类:
```typescript
export interface LayoutPosition {
  line: number;  // 0-based 布局行号
  col: number;   // 0-based 列号 (grapheme index in line)
}

export class TextLayoutEngine {
  private _text: string = "";
  private _width: number = 80;
  private _lines: string[] = [""];          // 布局后的行文本
  private _lineOffsets: number[] = [0];     // 每行起始 grapheme offset
  private _graphemes: string[] = [];        // 全文 grapheme 列表
  private _dirty: boolean = true;

  updateText(text: string): void;    // 设置文本, 标记 dirty
  updateWidth(width: number): void;  // 设置视口宽度, 标记 dirty

  getLineCount(): number;
  getLine(lineIndex: number): string;
  getGraphemeCount(): number;

  offsetToPosition(graphemeOffset: number): LayoutPosition;
  positionToOffset(position: LayoutPosition): number;
  getLayoutColumnFromOffset(offset: number): number;  // 布局列 (考虑 CJK 双宽)

  private _performLayout(): void;
  // 1. 使用 Intl.Segmenter('zh', { granularity: 'grapheme' }) 分割全文
  // 2. 按 \n 分段
  // 3. 每段内按 _width 换行 (使用 getCharWidth 计算每个 grapheme 的显示宽度)
  // 4. 记录 _lines 和 _lineOffsets
}
```

关键实现细节:
- Grapheme 分割使用 Bun 原生 `Intl.Segmenter`
- CJK 字符宽度=2，通过已有的 `getCharWidth` 计算
- 换行时如果一个 CJK 字符超出剩余宽度，整个字符换行到下一行
- `_lineOffsets[i]` 存储第 i 行第一个 grapheme 在全文 graphemes 数组中的 index

测试 15+ 用例:
- 空文本, 单行, 多行 (\n 分割)
- 超宽自动换行 (ASCII, CJK)
- offsetToPosition / positionToOffset 往返一致性
- Grapheme cluster: "e\u0301" (é), "👨‍👩‍👧" 各算 1 个 grapheme
- CJK 文本 "你好世界" 宽度正确

```typescript
export class TextEditingController {
  private _text: string = "";
  private _cursorPosition: number = 0;       // grapheme index
  private _preferredColumn: number = -1;      // 垂直移动记忆列
  private _vScrollOffset: number = 0;         // 垂直滚动偏移
  private _layoutEngine: TextLayoutEngine;
  private _listeners: Set<() => void> = new Set();
  private _disposed: boolean = false;

  constructor(options?: { text?: string; width?: number });

  // Getters
  get text(): string;
  get cursorPosition(): number;
  get lineCount(): number;
  get cursorLine(): number;
  get cursorColumn(): number;

  // 文本操作
  insertText(text: string): void;     // 在光标处插入, 更新 cursorPosition
  deleteText(count?: number): void;   // 向前删除 count 个 grapheme (Backspace)
  deleteForward(count?: number): void;// 向后删除 (Delete 键)

  // 光标移动
  moveCursorLeft(): void;
  moveCursorRight(): void;
  moveCursorUp(): void;              // 使用 preferredColumn
  moveCursorDown(): void;            // 使用 preferredColumn
  moveCursorToLineStart(): void;
  moveCursorToLineEnd(): void;
  moveCursorToStart(): void;         // 文档首
  moveCursorToEnd(): void;           // 文档尾

  // 布局相关
  updateWidth(width: number): void;
  getLayoutPosition(): LayoutPosition;  // 当前光标的行列位置

  // 监听器
  addListener(fn: () => void): void;
  removeListener(fn: () => void): void;
  private _notifyListeners(): void;

  dispose(): void;
}
```

关键实现:
- 所有位置计算基于 grapheme index (使用 TextLayoutEngine 的 grapheme 列表)
- `moveCursorUp/Down`: 先获取当前 LayoutPosition, 如果 _preferredColumn < 0 则记录当前布局列, 然后在目标行找到最接近 preferredColumn 的 grapheme offset
- `insertText`: 将 grapheme 列表在 cursorPosition 处 splice, 重建文本, cursorPosition += 新 grapheme 数
- `deleteText`: splice 移除 cursorPosition 前的 grapheme, cursorPosition -= count
- 水平移动重置 _preferredColumn = -1
