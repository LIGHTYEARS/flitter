extend 逻辑:
- 如果 extend=true 且无活跃选区 → selectionBase = 当前 cursorPosition
- 如果 extend=true → 移动后 selectionExtent = 新 cursorPosition
- 如果 extend=false (默认) 且有选区 → 清除选区 (base=extent=-1)

添加选区方法:
```typescript
deleteSelectedText(): void;          // 删除 [start, end) 范围
deleteSelectedOrText(count?: number): void;
selectWordAt(offset: number): void;  // 找到 offset 所在的词边界
selectLineAt(offset: number): void;  // 找到 offset 所在行的起止
setSelectionRange(start: number, end: number): void;
clearSelection(): void;
```

添加 Kill buffer 方法:
```typescript
deleteWordLeft(): void;       // 存入 killBuffer
deleteWordRight(): void;      // 存入 killBuffer
deleteToLineEnd(): void;      // Ctrl+K
deleteToLineStart(): void;    // Ctrl+U
deleteCurrentLine(): void;    // Ctrl+Shift+K
yankText(): void;             // Ctrl+Y
```

Kill buffer 追加逻辑:
- `_lastKillWasContiguous` 标记: 连续 kill 操作 (无其他编辑操作间隔) 时追加到 killBuffer
- 任何非 kill 操作 (insertText, deleteText, moveCursor 等) 重置 `_lastKillWasContiguous = false`
