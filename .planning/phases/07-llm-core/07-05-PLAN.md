## Gemini generateContent API (自行封装)

```typescript
/** POST https://generativelanguage.googleapis.com/v1beta/models/{model}:streamGenerateContent?alt=sse */
interface GeminiRequest {
  contents: GeminiContent[];
  systemInstruction?: { parts: GeminiPart[] };
  tools?: GeminiToolConfig[];
  generationConfig?: {
    maxOutputTokens?: number;
    temperature?: number;
    seed?: number;
    thinkingConfig?: {
      includeThoughts: boolean;
      thinkingLevel?: "MINIMAL" | "LOW" | "MEDIUM" | "HIGH";
    };
  };
  safetySettings?: GeminiSafetySetting[];
}

interface GeminiContent {
  role: "user" | "model";
  parts: GeminiPart[];
}

type GeminiPart =
  | { text: string }
  | { inlineData: { mimeType: string; data: string } }
  | { functionCall: { name: string; args: Record<string, unknown> } }
  | { functionResponse: { name: string; response: { content: string } } }
  | { thought: boolean; text: string };

interface GeminiToolConfig {
  functionDeclarations: GeminiFunctionDeclaration[];
}

interface GeminiFunctionDeclaration {
  name: string;
  description: string;
  parameters: Record<string, unknown>;
}

interface GeminiSafetySetting {
  category: string;
  threshold: string;
}

/** SSE 响应格式 (每个 chunk 是完整的 generateContentResponse) */
interface GeminiStreamChunk {
  candidates?: [{
    content: {
      role: "model";
      parts: GeminiPart[];
    };
    finishReason?: "STOP" | "MAX_TOKENS" | "SAFETY" | "RECITATION" | "OTHER";
  }];
  usageMetadata?: {
    promptTokenCount: number;
    candidatesTokenCount: number;
    totalTokenCount: number;
    cachedContentTokenCount?: number;
    thoughtsTokenCount?: number;
  };
}
```

2. 实现 `GeminiTransformer`:

**toProviderMessages(messages, systemPrompt)**:
  - systemInstruction: { parts: [{ text: joined systemPrompt }] }
  - contents: 遍历 Message[]:
    - UserMessage → role:"user", parts:
      - TextBlock → { text }
      - ImageBlock → { inlineData: { mimeType, data } }
      - ToolResultBlock → { functionResponse: { name, response:{ content } } }
    - AssistantMessage → role:"model", parts:
      - TextContentBlock → { text }
      - ToolUseBlock → { functionCall: { name, args } }
      - ThinkingBlock → { thought: true, text } (仅 provider="vertexai" 或无 provider)
    - InfoMessage → 跳过
  - Gemini 要求交替 user/model，合并相邻同角色

**fromProviderDelta(chunk: GeminiStreamChunk, state)**:
  - 每个 chunk 的 candidates[0].content.parts 是增量 parts
  - 遍历 parts:
    - { text, thought: true } → thinking block
    - { text } (无 thought) → text block
    - { functionCall } → tool_use block (完整的，非增量)
  - finishReason → 映射为 stopReason:
    - "STOP" → "end_turn"
    - "MAX_TOKENS" → "max_tokens"
    - 其他 → "end_turn"
  - usageMetadata → Usage:
    - promptTokenCount → inputTokens
    - candidatesTokenCount → outputTokens
    - cachedContentTokenCount → cacheReadInputTokens
    - thoughtsTokenCount → thinkingBudget
