## xAI ChatCompletion API (OpenAI 兼容格式)

```typescript
/** POST https://api.x.ai/v1/chat/completions */
interface XAIRequest {
  model: string;
  messages: XAIChatMessage[];
  tools?: XAITool[];
  stream: true;
  stream_options: { include_usage: true };
  top_p?: number;  // 默认 0.95
  tool_choice?: "auto" | "none" | { type: "function"; function: { name: string } };
}

type XAIChatMessage =
  | { role: "system"; content: string }
  | { role: "user"; content: string | XAIContentPart[] }
  | { role: "assistant"; content: XAIContentPart[] | null; tool_calls?: XAIToolCall[] }
  | { role: "tool"; tool_call_id: string; content: string };

type XAIContentPart =
  | { type: "text"; text: string };
  // 注意: xAI 不支持图片内容

interface XAIToolCall {
  id: string;
  type: "function";
  function: { name: string; arguments: string };
}

interface XAITool {
  type: "function";
  function: {
    name: string;
    description: string;
    parameters: Record<string, unknown>;
  };
}

/** SSE 响应格式 (OpenAI ChatCompletion 兼容) */
interface XAIStreamChunk {
  id: string;
  object: "chat.completion.chunk";
  model: string;
  choices: [{
    index: number;
    delta: {
      role?: "assistant";
      content?: string | null;
      tool_calls?: [{
        index: number;
        id?: string;
        type?: "function";
        function?: {
          name?: string;
          arguments?: string;
        };
      }];
    };
    finish_reason: "stop" | "tool_calls" | "length" | null;
  }];
  usage?: {
    prompt_tokens: number;
    completion_tokens: number;
    total_tokens: number;
  };
}
```

1. 定义 xAI ChatCompletion 原生类型
2. 实现 `XAITransformer` 类继承 `BaseMessageTransformer`:

**toProviderMessages(messages, systemPrompt)**:
  - system: systemPrompt → { role:"system", content: joined text }
  - messages: 遍历 Message[]:
    - UserMessage → role:"user", content:
      - TextBlock → { type:"text", text }
      - ImageBlock → **跳过** (xAI 不支持图片)
      - ToolResultBlock → { role:"tool", tool_call_id: id.replace(/^toolu_/, ""), content }
    - AssistantMessage → role:"assistant":
      - TextContentBlock → content 数组 { type:"text", text }
      - ToolUseBlock → tool_calls: [{ id: id.replace(/^toolu_/, ""), type:"function", function: { name, arguments } }]
      - ThinkingBlock → 跳过 (xAI 无 thinking 支持)
    - InfoMessage → 特殊处理:
      - manual_bash_invocation → user 消息
      - 其他 → 跳过
  - 合并相邻同角色消息

**fromProviderDelta(chunk: XAIStreamChunk, state)**:
  - 遍历 choices[0].delta:
    - delta.content (string) → text block 增量
    - delta.tool_calls → tool_use block:
      - id + function.name 存在 → 新建 tool_use block
      - function.arguments 存在 → 追加 JSON 增量
    - choices[0].finish_reason:
      - "stop" → stopReason "end_turn"
      - "tool_calls" → stopReason "tool_use"
      - "length" → stopReason "max_tokens"
  - usage (仅最后一个 chunk):
    - prompt_tokens → inputTokens
    - completion_tokens → outputTokens
    - cacheCreationInputTokens: null
    - cacheReadInputTokens: null
  </action>
