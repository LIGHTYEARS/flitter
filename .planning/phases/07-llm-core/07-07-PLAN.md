<interfaces>
## Provider Registry API

```typescript
/**
 * 创建 Provider 实例
 * @throws ProviderError 如果 name 不是已知的 Provider
 */
function createProvider(name: ProviderName): LLMProvider;

/**
 * 从模型 ID 查找 ModelInfo
 * @returns ModelInfo | undefined
 */
function resolveModel(model: string): ModelInfo | undefined;

/**
 * 从模型名推断 Provider
 * 优先查 MODEL_REGISTRY，fallback 到前缀匹配:
 *   - "claude-*" | "anthropic/*" → "anthropic"
 *   - "gpt-*" | "o3*" | "o4*" | "codex-*" | "openai/*" → "openai"
 *   - "gemini-*" | "vertexai/*" → "gemini"
 *   - "grok-*" | "xai/*" → "xai"
 *
 * @throws ProviderError 如果无法推断
 */
function resolveProvider(model: string): ProviderName;

/**
 * 一站式: 从模型名获取 Provider 实例
 */
function getProviderForModel(model: string): LLMProvider;
```
</interfaces>

      - gpt-*, o3*, o4*, codex-*, openai/* → "openai"
      - gemini-*, vertexai/* → "gemini"
      - grok-*, xai/* → "xai"
   d. 无法推断 → throw ProviderError

5. 实现 `getProviderForModel(model: string): LLMProvider`:
