### Task 7: Unify error handling (PROTO-07)

**File:** `packages/flitter-amp/src/state/app-state.ts`

Add `handleError(message: string)` method:
```typescript
handleError(message: string): void {
  this.conversation.finalizeAssistantMessage();
  this.conversation.finalizeThinking();
  this.conversation.isProcessing = false;
  this.error = message;
  this.notifyListeners();
}
```

**File:** `packages/flitter-amp/src/index.ts`

Update `handleSubmit` catch block to call `appState.handleError(message)` instead of manually setting state.

**Verification:** All error paths converge through `handleError()`. No orphaned state after errors.
