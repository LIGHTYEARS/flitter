## 实现细节

### 1. sse-parser.ts — SSE 事件解析 TransformStream

```ts
interface SSEEvent {
  event?: string;
  data: string;
  id?: string;
}

// TransformStream<string, SSEEvent>

// 多行 data 用 \n 拼接
// 忽略注释 (: 前缀)
```

### 2. streamable-http.ts — StreamableHTTP 传输

关键流程:
```
send(msg):
  1. POST JSON to url
     Headers: Content-Type=application/json, Accept=application/json+text/event-stream
     + mcp-session-id (if captured)
     + mcp-protocol-version (if set)
     + Authorization (from authProvider)
  2. Check response status:
     - 401 → re-auth flow
     - 202 → accepted (notification)
     - 200 → check Content-Type
  3. If text/event-stream → pipe through TextDecoderStream → SSEEventParser → parse each data as JSON-RPC
  4. If application/json → parse body as JSON-RPC
  5. Capture mcp-session-id from response headers
```

### 3. 测试策略

使用 Node.js `http.createServer` 作为 mock MCP server:
- 基本通信: POST → JSON 响应
