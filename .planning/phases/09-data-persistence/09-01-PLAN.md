## 实现内容

### 1. `packages/data/src/thread/types.ts`

```typescript
// ThreadEntry — 线程索引条目（轻量摘要，不含完整消息）
interface ThreadEntry {
  id: string;
  v: number;
  created: number;
  title: string | null;
  userLastInteractedAt: number;
  messageCount: number;
  env?: ThreadEnvironment;
  originThreadID?: string;
  mainThreadID?: string;
  relationships: ThreadRelationship[];
  summaryStats: {
    diffStats?: Record<string, unknown>;
    messageCount: number;
  };
  agentMode?: string;
  usesDtw: boolean;
  archived?: boolean;
  creatorUserID?: string;
  meta?: { visibility?: string; sharedGroupIDs?: string[] };
}

// ThreadStoreOptions
interface ThreadStoreOptions {
  maxThreads?: number;          // null = unlimited (default)
  uploadThrottleMs?: number;    // 用于持久化节流 (default: 1000)
}
```

### 2. `packages/data/src/thread/thread-store.ts`

ThreadStore class 功能:
- `constructor(options?: ThreadStoreOptions)` — 初始化
- `setCachedThread(thread: ThreadSnapshot, opts?)` — 缓存线程 + 更新 ThreadEntry 索引 + 标记 dirty
- `getThread(id: string)` — 获取缓存线程 (返回 BehaviorSubject)
- `deleteThread(id: string)` — 从缓存和索引中删除
- `observeThread(id: string)` — 返回 Observable<ThreadSnapshot>
- `observeThreadEntries()` — 返回排序后的 ThreadEntry 列表 Observable
- `markDirty(id: string)` — 标记线程待持久化
- `getDirtyThreadIds()` — 获取所有 dirty 线程 ID
- `clearDirty(id: string)` — 清除 dirty 标记

内部:
- `threadSubjects: Map<string, BehaviorSubject<ThreadSnapshot>>`
- `threadEntriesByID: Map<string, ThreadEntry>`
- `threadEntriesState: BehaviorSubject<ThreadEntry[] | null>`
- `dirtyThreads: Set<string>`
- `snapshotToEntry(thread: ThreadSnapshot): ThreadEntry` — 从 fuT 翻译
- `entryEquals(a: ThreadEntry, b: ThreadEntry): boolean` — 从 T4 翻译
- `computeUserLastInteractedAt(thread: ThreadSnapshot): number` — 从 HqR 翻译

### 3. `packages/data/src/thread/thread-store.test.ts`

测试用例 (~25 tests):
- **创建**: 新线程加入缓存，ThreadEntry 自动生成
- **读取**: getThread 返回正确快照，不存在返回 undefined
- **更新**: setCachedThread 更新已有线程，版本号递增
- **删除**: deleteThread 清除缓存和索引
- **Dirty tracking**: markDirty/clearDirty/getDirtyThreadIds
- **ThreadEntry 索引**: 按 userLastInteractedAt 降序排列
- **ThreadEntry 相等性**: entryEquals 正确比较
- **maxThreads 限制**: entries 数量不超过上限
- **空线程自动删除**: messages.length === 0 且无 draft 时自动从索引中删除
- **Observable**: observeThread 和 observeThreadEntries 发出正确值
