## Test Infrastructure

### KeyEvent Factory

Use `createKeyEvent()` from `flitter-core/src/input/events` which is already established in the test suite:

```typescript
import { createKeyEvent } from 'flitter-core/src/input/events';

const ctrlC = createKeyEvent('c', { ctrlKey: true });
const altT = createKeyEvent('t', { altKey: true });
const esc = createKeyEvent('Escape');
```

### AppState Test Factory

Use `AppState.create()` with a mock provider, following the pattern from `__tests__/app-state.test.ts`:

```typescript
const provider: Provider = {
  model: 'test-model',
  async *sendPrompt() { /* ... */ },
  cancelRequest() {},
};
const appState = AppState.create({ cwd: '/tmp', provider });
```

### Widget Tree Testing

For widget tree assertions, create the InputArea widget and inspect its `build()` output:

```typescript
const inputArea = new InputArea({
  onSubmit: () => {},
  isProcessing: false,
  mode: null,
});
const state = inputArea.createState() as InputAreaState;
// Mount and verify build output
```
