Per D-01 and D-02, add gap-aware border rendering to PaintContext:

1. **Define `BorderGap` interface** above `drawBorder` method:
```typescript
export interface BorderGap {
  start: number;
  end: number;
}
```
Where `start` is column offset from border left edge (inclusive, 0-based where 0 = left corner column), `end` is exclusive. Gaps must only affect columns between corners (col 1 through col w-2).

2. **Define `BorderGaps` interface**:
```typescript
export interface BorderGaps {
  top?: BorderGap[];
  bottom?: BorderGap[];
}
```

3. **Modify `drawBorder` signature** to accept optional gaps:
```typescript
drawBorder(
  x: number, y: number, w: number, h: number,
  borderStyle: BoxDrawingStyle, color?: Color,
  gaps?: BorderGaps,
): void
```
In the top horizontal edge loop (`for (let col = x + 1; col < x + w - 1; col++)`), add gap check:
```typescript
const topGaps = gaps?.top ?? [];
for (let col = x + 1; col < x + w - 1; col++) {
  const relCol = col - x;
  if (topGaps.some(g => relCol >= g.start && relCol < g.end)) continue;
  this.drawChar(col, y, chars.h, style, 1);
}
```
Same for bottom horizontal edge with `gaps?.bottom`.

CRITICAL: The existing top/bottom loop draws BOTH top and bottom in the same loop iteration. Split into separate loops:
- Top edge: `for (let col = x + 1; col < x + w - 1; col++)` with top gaps check
- Bottom edge: `for (let col = x + 1; col < x + w - 1; col++)` with bottom gaps check

4. **Modify `drawBorderSides` signature** to accept optional gaps:
```typescript
drawBorderSides(
  x: number, y: number, w: number, h: number,
  sides: { ... },
  gaps?: BorderGaps,
): void
```
In the fast path (all sides same), pass gaps through to `drawBorder`:
```typescript
if (sameStyle && sameColor) {
  this.drawBorder(x, y, w, h, topStyle, topColor, gaps);
  return;
}
```
In the slow path top/bottom loops, add gap skipping identical to drawBorder.

5. **Backward compatibility**: Both gaps parameters default to undefined. When undefined, behavior is identical to current implementation. No existing callers need changes.

```typescript
export function lerpColor(a: Color, b: Color, t: number): Color {
  const clamped = Math.max(0, Math.min(1, t));
  const rgbA = a.toRgb();
  const rgbB = b.toRgb();
  if (rgbA.mode !== 'rgb' || rgbB.mode !== 'rgb') {
    return clamped < 0.5 ? a : b;
  }
  const r = Math.round(rgbA.r + (rgbB.r - rgbA.r) * clamped);
  const g = Math.round(rgbA.g + (rgbB.g - rgbA.g) * clamped);
  const bl = Math.round(rgbA.b + (rgbB.b - rgbA.b) * clamped);
  return Color.rgb(r, g, bl);
}
```
