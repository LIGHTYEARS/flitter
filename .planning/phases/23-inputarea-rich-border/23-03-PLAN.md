**Part A: BorderShimmer widget (D-14, D-15)**

Create `packages/flitter-cli/src/widgets/border-shimmer.ts`:

```typescript
import { StatefulWidget, State, Widget, ... } from 'flitter-core';
import { lerpColor, Color, ... } from 'flitter-core';
```

**BorderShimmer props:**
```typescript
interface BorderShimmerProps {
  color: Color;           // target mode color (e.g., smartModeColor)
  backgroundColor: Color; // border default color (e.g., brightBlack)
  trigger: number;        // agentModePulseSeq — animation starts when this changes
  trail: number;          // trail length in characters (default 5, per D-14)
  direction: 'right-to-left' | 'left-to-right';  // 'right-to-left' per D-14
  borderStyle: BoxDrawingStyle;  // to draw colored horizontal chars
}
```

**BorderShimmerState:**
- Private fields: `_ticker: Ticker | null`, `_position: number`, `_active: boolean`, `_lastTrigger: number`, `_totalWidth: number`
- `initState()`: Store initial trigger value, do not start animation
- `didUpdateWidget(old)`: If `trigger` changed, start animation (set `_active = true`, `_position = _totalWidth` for right-to-left, create/start ticker)
- `dispose()`: Stop and dispose ticker (D-14 pitfall: always clean up)
- `_onTick(elapsed)`: Advance `_position` by `speed` (2 chars per tick). If `_position < -trail` (for right-to-left), stop animation. Call `setState`.
- Speed: 2 characters per frame tick (Claude's Discretion — ~3.3 second sweep on 200-char border at 30fps)
- FPS: Use Ticker's natural frame rate (synchronized with FrameScheduler)

**BorderShimmerState.build():**
- If `!_active`, return `SizedBox({ height: 1 })` (invisible)
- Use `LayoutBuilder` or similar to get available width → set `_totalWidth`
- Create a `CustomPaint`-style widget (or override `createRenderObject`) that paints:
  - For each column `col` in 0..totalWidth:
    - Distance from head = `|col - _position|` (adjusted for direction)
    - If distance < trail: compute `t = 1 - distance / trail`, use `lerpColor(backgroundColor, color, t)` to get column color
    - Draw `\u2500` (horizontal border char) at that column with the computed color
    - If distance >= trail: draw nothing (gap is transparent — underlying border char shows through)
- This means the shimmer only overwrites border characters within its trail radius, leaving the rest alone
