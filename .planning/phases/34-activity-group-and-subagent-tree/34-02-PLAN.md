# Plan 34-02 — Subagent Message Nesting and Task Label (Wave 2)

**Requirements:** ACTV-04, ACTV-05
**Wave:** 2 (depends on 34-01)

## Objective

Wire inline subagent text messages into ActivityGroup as text-kind actions,
ensure TaskTool uses "Subagent" label prefix matching AMP, and add streaming
progress display for active subagent groups.

## Files

### Modified
- `packages/flitter-cli/src/widgets/tool-call/task-tool.ts` — Update to use "Subagent" display name
- `packages/flitter-cli/src/widgets/activity-group.ts` — Add inline text message rendering
- `packages/flitter-cli/src/widgets/chat-view.ts` — Wire subagent text messages into ActivityGroup

## Implementation Steps

### Step 1: Update TaskTool display name to "Subagent"

In task-tool.ts, the header name should be "Subagent" (matching AMP golden):
- AMP shows: `⢠ Subagent Explore flitter-cli package structure ▼`
- The resolveToolDisplayName already returns "Task" for these - update to "Subagent"

Update resolve-tool-name.ts TOOL_NAME_MAP:
```
Task: 'Subagent',
oracle: 'Subagent',
code_review: 'Subagent',
librarian: 'Subagent',
```

### Step 2: Wire subagent inline messages into ActivityGroup

In chat-view.ts, when building an ActivityGroup for a Task tool call:
- Include subagent text messages (from tool's streaming output) as text-kind actions
- These appear as `├── Some inline text from the subagent` in the tree
- AMP shows these as plain text interleaved with tool calls in the tree

### Step 3: Add streaming progress to ActivityGroup header

When the group is active and streaming:
- Header shows BrailleSpinner + "Subagent" bold + description dim
- Matches AMP: `⣒ Subagent Explore flitter-cli package structure ▼`
- The description comes from Task tool's rawInput (Description/description/Prompt/prompt)
