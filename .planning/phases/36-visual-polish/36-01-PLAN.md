## Changes

### 1. VPOL-01: Block cursor with 500ms blink

**File:** `packages/flitter-cli/src/widgets/streaming-cursor.ts`

- Change `CURSOR_CHAR` from `\u258C` (▌ LEFT HALF BLOCK) to `\u2588` (█ FULL BLOCK)
- Change `CURSOR_BLINK_INTERVAL_MS` from 530 to 500
- Update doc comments to reflect new character and interval

### 2. VPOL-02: OSC8 terminal hyperlinks for file paths

**New utility:** `packages/flitter-cli/src/utils/osc8-link.ts`

- Export `wrapOSC8Link(text: string, url: string): string` that wraps text in `\x1b]8;;{url}\x07{text}\x1b]8;;\x07`
- Export `fileLink(filePath: string, displayText?: string): string` that builds a `file://` URL and calls wrapOSC8Link

**File:** `packages/flitter-cli/src/widgets/tool-call/tool-header.ts`

- Import `fileLink` from osc8-link utility
- In detail span rendering, detect file path patterns and wrap in OSC8 hyperlink
- The detail text already contains the shortened path; wrap the TextSpan text with the OSC8 sequence using the full file path as the link URL

### 3. VPOL-03: Diff stats in collapsed edit file header

**File:** `packages/flitter-cli/src/widgets/tool-call/edit-file-tool.ts`

- Move `countDiffStats()` call to occur before the expansion check
- Extract diff stats and compute the "+N/-M" string
- Pass the stats string as an additional detail to ToolHeader (e.g., details: [shortenPath(filePath), '+5/-3'])
- This ensures collapsed headers show diff summary without needing expansion

### 4. VPOL-04: Idle prompt symbol ">"
