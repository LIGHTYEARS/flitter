# Plan 38-03: Agentic loop instrumentation

<objective>
Instrument the agentic loop in prompt-controller.ts with trace spans (agent, prompt-assembly, inference, tools, tool:{name}) and add structured error capture. Enrich UsageInfo with cache token fields and measure TTFT.
</objective>

<must_haves>
- Root "agent" span wraps every submitPrompt() call
- "prompt-assembly" span wraps _buildMessages()
- "inference" span wraps _streamResponse() with TTFT event and usage attributes
- "tools" container span wraps _executeToolCalls()
- Individual "tool:{name}" spans per tool execution
- Structured error capture in all catch blocks with traceId/spanId context
- UsageInfo enriched with cacheReadTokens and cacheWriteTokens
- Tracing failures NEVER break the agentic loop (all tracing in try/catch)
</must_haves>

<tasks>
